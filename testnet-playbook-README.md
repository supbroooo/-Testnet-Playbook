## 🚀 Getting Started (Complete Beginner)

### Step 1 — Set Up a Wallet
- Download [MetaMask](https://metamask.io) or [Phantom](https://phantom.app)
- Create a new wallet (use a fresh one for testnets)
- Save your seed phrase somewhere safe offline

### Step 2 — Add the Testnet Network
- Go to [chainlist.org](https://chainlist.org)
- Search for the testnet
- Click "Add to MetaMask"

### Step 3 — Get Testnet Tokens
- Visit the faucet link in the `faucets/` folder
- Paste your wallet address
- Receive free testnet tokens

### Step 4 — Complete Tasks
- Follow the guide in `guides/` for your chosen testnet
- Check off tasks in the checklist
- Screenshot everything for proof
